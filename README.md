# SaltSense AI

Adaptive multisensory taste modulation system.

## Overview
SaltSense AI is a hardware-software system designed to alter salt perception using electrical stimulation, temperature, aroma, and adaptive algorithms.

## Features
- Electrical taste modulation (DAC controlled)
- Temperature-based enhancement
- Aroma-assisted perception
- Bluetooth control (ESP32)
- Modular hardware design

## Hardware
- ESP32
- MCP4725 DAC
- Op-Amp buffer
- Heating module
- Aroma module

## Warning
This is a research prototype and not a medical device.
